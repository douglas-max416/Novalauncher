# ✦ NovaLauncher

A beautiful One UI 9-inspired Android launcher with liquid glass effects, smooth animations, and zero ads.

---

## 📲 How to Get Your APK (Android Only — No PC Needed)

### Step 1 — Upload to GitHub
1. Go to **github.com** and log in
2. Click the **+** button → **New repository**
3. Name it `NovaLauncher`, set to **Public**, click **Create repository**
4. Click **uploading an existing file**
5. Extract the ZIP on your phone and upload ALL files maintaining the folder structure
   - Tip: Use a file manager app to extract the ZIP first

### Step 2 — Trigger the Build
1. Once files are uploaded, click **Actions** tab in your repo
2. You'll see **"Build NovaLauncher APK"** workflow
3. Click it → click **"Run workflow"** → click the green **"Run workflow"** button
4. Wait ~5 minutes for it to build ⏳

### Step 3 — Download Your APK
1. When the build turns ✅ green, click on it
2. Scroll to **Releases** section on your repo main page
3. Download `app-debug.apk` directly to your phone
4. Open the APK → install it
5. Set **Nova Launcher** as your default home app when prompted 🎉

## ✨ Features
- 🌊 Liquid glass animated background
- 🕐 Clock + Date + Battery + Weather widgets
- 📱 Scrollable home screen pages with smooth page transitions
- 🔍 App search bar
- 🚢 Glass pill dock (4–6 apps)
- 👁️ Hidden apps manager
- 🖼️ Custom wallpaper support
- 👋 First-launch welcome screen
- 🚫 Zero ads, ever

## 👆 Gestures
| Gesture | Action |
|---|---|
| Swipe up | Open app drawer |
| Swipe down | Pull notification shade |
| Long press desktop | Settings / Wallpaper / Refresh |
| Long press icon | Hide / Dock / App Info |
| Double tap | Lock screen |

## 🛠 Tech Stack
- Kotlin + AndroidX
- ViewPager2 for scrollable pages
- Custom Canvas drawing for glass effects
- LiveData + ViewModel architecture
- No third-party ad SDKs
